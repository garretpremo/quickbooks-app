Privacy Policy

Last updated: February 8, 2026

Overview

This application is a personal/internal tool that connects to QuickBooks Online via the Intuit API. It is not a public-facing application.

Data Collection & Use

  • This application accesses QuickBooks Online data solely on behalf of the authorized account owner.
  • Data retrieved from QuickBooks Online is used exclusively for the account owner's own business purposes.
  • No data is collected, stored, or transmitted to any third party.
  • API credentials and access tokens are stored locally on the account owner's machine.

Data Security

  • All communication with QuickBooks Online uses HTTPS encryption.
  • OAuth 2.0 is used for authentication and authorization.
  • Access tokens are stored locally and are never shared.

Contact

For questions about this policy, contact the application owner directly.
